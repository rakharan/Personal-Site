# Personal Site — Rakha Randhikatama

Personal portfolio website built with Next.js, TypeScript, and Tailwind CSS. Showcases selected projects, resume, and photography work.

Live site: [sagameda.com](https://sagameda.com)

## Tech Stack

- **Framework:** Next.js (Pages Router, static export)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** AOS (Animate On Scroll)
- **Icons:** React Icons

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
```

The static output is exported to `out/` and deployed via FTP.

## Project Structure

```
src/
├── components/     # Reusable UI components
├── pages/          # Next.js pages (routes)
├── styles/         # Global CSS
└── assets/         # Static assets (images, fonts)
```
